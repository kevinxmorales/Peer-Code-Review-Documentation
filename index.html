Peer Code Review Documentation

Introduction
The Peer Code Review Platform is a web application built to help SUNY Oswego students become better programmers. The platform consists of a React frontend, Node JS and Spring Boot web services, and Postgres SQL database.
Installation
This section will cover how to get started with development on the platform. We'll cover setting up local development environment, running the application locally, and deploying to production.
User Interface
The user interface is a React application built with the NextJS framework and written in TypeScript.
Web Services
The platform uses a multi-tiered approach for server side technologies. This includes a Java Spring Boot REST API which handles the bulk of the platforms operations, and a Node JS API, which handles authentication.
Database
For persistent data storage, the platform utilizes a Postgres SQL database.
